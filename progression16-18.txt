Progression การเรียนรู้สัปดาห์นี้: Cookie & Form Settings Project

สัปดาห์นี้ผมได้ฝึกทำสองโปรเจกต์หลักที่เกี่ยวข้องกับการจัดการข้อมูลฝั่งผู้ใช้ในเบราว์เซอร์ ได้แก่ Cookie Management Project และ 
Form Settings Project ซึ่งทั้งสองงานช่วยให้เข้าใจกลไกของการเก็บข้อมูลแบบ persistent ใน browser ทั้งจาก document.cookie และ localStorage 
รวมถึงการเชื่อมโยง DOM และ Event เข้ากับการทำงานของ JavaScript

เริ่มต้นจากโปรเจกต์ Cookie Management ผมได้ฝึกอ่านและเขียนคุกกี้โดยตรงผ่าน document.cookie เพื่อเข้าใจรูปแบบ key=value และการคั่นด้วย ; จากนั้นสร้างคลาส
CookieUtil ที่มีเมธอด get(), set(), และ unset() เพื่อใช้จัดการคุกกี้อย่างเป็นระบบ เรียนรู้การใช้ encodeURIComponent และ decodeURIComponent เพื่อป้องกันอักขระพิเศษ
การตั้งวันหมดอายุด้วย Date() และหลักการ “ลบคุกกี้ด้วยการเขียนทับและตั้งวันหมดอายุในอดีต” นอกจากนี้ยังได้ฝึกเชื่อมโยงไฟล์ด้วย import/export แบบ ES Module
เพื่อให้โค้ดมีโครงสร้างที่ชัดเจนและเป็นมาตรฐาน

ต่อมาในโปรเจกต์ Form Settings ผมได้ฝึกใช้ DOM และ Event Handling เพื่อควบคุมการตั้งค่าหน้าจอ เช่น สีพื้นหลัง สีตัวอักษร และขนาดฟอนต์ โดยบันทึกข้อมูลเหล่านี้ลงใน localStorage 
เพื่อให้คงอยู่แม้ผู้ใช้รีเฟรชหน้าเว็บ เรียนรู้การใช้ localStorage.setItem() และ getItem() ควบคู่กับการอัปเดต DOM แบบเรียลไทม์ นอกจากนี้ยังได้ฝึกสร้างฟังก์ชัน applyFontSize() 
เพื่อจัดการขนาดตัวอักษรอย่างเป็นระบบ และใช้ปุ่ม Reset เพื่อเคลียร์ค่าทั้งหมดด้วย localStorage.clear() พร้อมรีเซ็ต UI กลับสู่ค่ามาตรฐาน

สิ่งที่ได้เรียนรู้จากสัปดาห์นี้คือ

เข้าใจความแตกต่างระหว่าง Cookies และ localStorage

ฝึกการเชื่อมโยง DOM, Event, และการจัดเก็บข้อมูลฝั่ง client

ได้ฝึกออกแบบ Utility Class ที่ reusable

ได้พัฒนาทักษะการเขียน ES Module และการดีบักปัญหา scope ของโมดูล

เข้าใจหลักการ persistence ของข้อมูลใน browser และการอัปเดต UI แบบเรียลไทม์
