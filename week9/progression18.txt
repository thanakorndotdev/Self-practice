Progression การเรียนรู้: Music List Project

เริ่มต้นจากการเรียนรู้ DOM Manipulation
ผมเริ่มจากการใช้คำสั่ง document.addEventListener("DOMContentLoaded", …) เพื่อให้โค้ดรันหลังจากหน้าเว็บโหลดเสร็จ จากนั้นใช้ 
document.body.style.backgroundColor เพื่อเปลี่ยนสีพื้นหลังของเว็บ และใช้ prompt() เพื่อรับชื่อจากผู้ใช้แล้วแสดงผลบนหน้าเว็บด้วย 
querySelector('h1') และ .textContent ขั้นตอนนี้ช่วยให้เข้าใจพื้นฐานของการเข้าถึงและจัดการองค์ประกอบในหน้า HTML ด้วย JavaScript

เข้าใจการเก็บข้อมูลด้วย Array และ Object
จากนั้นผมเริ่มสร้างตัวแปร musics เป็น array สำหรับเก็บข้อมูลของเพลงแต่ละเพลงในรูปแบบ object ที่มี key เป็น Name และ Author 
พร้อมใช้คำสั่ง push() เพื่อเพิ่มข้อมูลใหม่เข้าไปใน array ทำให้เข้าใจแนวคิดของการเก็บข้อมูลหลายชุดในโครงสร้างที่เป็นระบบ

ฝึกออกแบบฟังก์ชันให้ทำงานเฉพาะด้าน (Modular Design)
ผมแบ่งโค้ดออกเป็นฟังก์ชันย่อย เช่น addMusic() สำหรับเพิ่มเพลงใหม่, getAllMusics() สำหรับเรียกดูข้อมูลทั้งหมด และ renderMusics() 
สำหรับแสดงผลในหน้าเว็บ การออกแบบแบบนี้ช่วยให้โค้ดอ่านง่าย ดูแลรักษาง่าย และสามารถเรียกใช้ซ้ำได้ในอนาคต

ฝึกเชื่อมข้อมูลเข้ากับ DOM แบบไดนามิก
ผมใช้ document.getElementById('music-list') เพื่อเข้าถึงตำแหน่งที่ต้องการแสดงเพลง และใช้ createElement('p') ร่วมกับ 
appendChild() เพื่อสร้างและเพิ่มแท็ก <p> สำหรับแต่ละเพลงในหน้าเว็บ การลูปผ่านข้อมูลใน array ทำให้เห็นภาพชัดเจนว่า JavaScript 
สามารถสร้างเนื้อหาบนหน้าเว็บได้แบบอัตโนมัติและยืดหยุ่น

การรวมทุกแนวคิดเข้าด้วยกัน (Integration)
เมื่อรวมทุกส่วนเข้าด้วยกัน หน้าเว็บของผมสามารถเปลี่ยนสีพื้นหลัง แสดงชื่อของผู้ใช้ และสร้างรายการเพลงพร้อมศิลปินได้อย่างสมบูรณ์แบบโดยอัตโนมัติในครั้งเดียว 
แสดงให้เห็นถึงความเข้าใจในการทำงานร่วมกันของข้อมูล (Data) และการแสดงผล (UI)

แนวทางพัฒนาในขั้นต่อไป
ขั้นต่อไปสามารถต่อยอดได้โดยการเพิ่มช่องกรอกเพลงใหม่และปุ่ม “Add Music” เพื่อให้ผู้ใช้เพิ่มเพลงได้เอง เพิ่มการบันทึกข้อมูลด้วย 
LocalStorage เพื่อให้ข้อมูลไม่หายเมื่อรีเฟรช และปรับปรุงหน้าตาเว็บไซต์ด้วย CSS หรือ Tailwind เพื่อให้สวยงามและใช้งานง่ายขึ้น
