Progression การเรียนรู้: Form Settings Project

เริ่มจากทำความเข้าใจกับโครงสร้างของไฟล์ index.html และ main.js โดยไฟล์ HTML ใช้สร้างฟอร์มให้ผู้ใช้สามารถเลือกสีพื้นหลัง 
สีตัวอักษร และขนาดฟอนต์ ส่วน main.js ใช้ควบคุมการบันทึกและดึงค่าที่เก็บไว้ใน localStorage เพื่อให้การตั้งค่าของผู้ใช้คงอยู่แม้รีเฟรชหน้าเว็บใหม่

ขั้นแรก ฝึกพื้นฐาน DOM และ Events โดยใช้ document.getElementById() เพื่อเข้าถึง element ต่าง ๆ เช่น input และ select พร้อมผูก event
listener ให้กับปุ่ม Save และ Reset ฝึกใช้ addEventListener("click", …) เพื่อให้เกิดการตอบสนองเมื่อผู้ใช้กดปุ่มในฟอร์ม

ขั้นต่อมา ฝึกใช้ localStorage เพื่อบันทึกค่าการตั้งค่าจากผู้ใช้ เช่น สีพื้นหลัง สีตัวอักษร และขนาดตัวอักษร ด้วย localStorage.setItem() และนำค่าที่บันทึกไว้กลับมาใช้หลังรีเฟรชด้วย 
localStorage.getItem() ได้เข้าใจหลักการเก็บข้อมูลถาวรฝั่ง client และการอ่านค่ากลับมาปรับ UI ให้ตรงกับการตั้งค่าก่อนหน้า

จากนั้นฝึกสร้างฟังก์ชัน applyFontSize() เพื่อควบคุมขนาดตัวอักษรให้เหมาะสมกับค่าที่ผู้ใช้เลือก เช่น small = 14px, medium = 18px, large = 22px 
เพื่อให้การเปลี่ยนขนาดตัวอักษรเป็นระบบและเข้าใจหลักการของการใช้เงื่อนไข if-else ในการควบคุมการทำงานของฟังก์ชัน

ขั้นถัดไป ฝึกเพิ่มปุ่ม Reset เพื่อให้ผู้ใช้สามารถล้างค่าทั้งหมดที่เคยตั้งไว้ โดยใช้ localStorage.clear() และรีเซ็ตค่าเริ่มต้นกลับเป็นพื้นหลังขาว ตัวอักษรดำ 
ฟอนต์ขนาดเล็ก ฝึกจัดการ DOM หลายจุดพร้อมกันให้กลับสู่ค่ามาตรฐาน

เมื่อเข้าใจทั้งหมดแล้ว ฝึกจัดการลำดับเหตุการณ์และการแสดงผลแบบเรียลไทม์ เช่น การอัปเดตสีและขนาดตัวอักษรทันทีหลังจากผู้ใช้กดบันทึก เพื่อให้เข้าใจหลักการเชื่อมต่อระหว่าง 
Event, DOM Manipulation และการจัดเก็บข้อมูลถาวรในเบราว์เซอร์

สุดท้าย สรุปสิ่งที่ได้เรียนรู้จากโปรเจกต์นี้คือ การใช้งาน localStorage เพื่อบันทึกข้อมูลผู้ใช้ การเชื่อมโยง DOM กับ JavaScript เพื่ออัปเดตส่วนติดต่อผู้ใช้แบบเรียลไทม์ การใช้ 
event listener หลายตัวในหน้าเดียวกัน และการออกแบบฟังก์ชันให้ reusable สำหรับควบคุมค่าที่ซ้ำใช้บ่อยในระบบ
