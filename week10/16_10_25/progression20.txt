Progression การเรียนรู้: Cookie Management Project

เริ่มจากทำความเข้าใจกับโครงสร้างของทั้งสามไฟล์คือ index.html, main.js และ cookieUtil.js โดยลองรันโค้ด 
สังเกตผลลัพธ์ของ document.cookie และทดสอบการสร้าง อ่าน และลบคุกกี้ เพื่อให้เข้าใจภาพรวมก่อนเริ่มฝึกอย่างเป็นขั้นตอน

ขั้นแรก ฝึกพื้นฐานการจัดการคุกกี้โดยตรงจาก document.cookie เพื่อเรียนรู้รูปแบบ key=value และการคั่นด้วยเครื่องหมาย “;” 
รวมถึงการตั้งวันหมดอายุด้วย expires จากนั้นเริ่มเขียนคลาส CookieUtil เพื่อรวมคำสั่งทั้งหมดให้อยู่ในรูปแบบที่เรียกใช้งานง่ายขึ้นผ่านเมธอด static

ต่อมา ฝึกเขียนเมธอด get(), set(), และ unset() ภายในคลาส CookieUtil โดย get() ใช้ค้นหาและดึงค่าคุกกี้ออกจากสตริงหลัก set() 
ใช้สร้างคุกกี้ใหม่พร้อมกำหนดวันหมดอายุ และ unset() ใช้ลบคุกกี้ด้วยการตั้งวันหมดอายุเป็นอดีต ฝึกใช้ encodeURIComponent และ decodeURIComponent 
เพื่อป้องกันอักขระพิเศษและให้แน่ใจว่าคุกกี้ถูกเข้ารหัสถูกต้อง

หลังจากเข้าใจพื้นฐานแล้ว ทดลองเรียกใช้งานใน main.js เพื่อตรวจสอบผลการทำงานของแต่ละเมธอด เช่น การสร้างคุกกี้ใหม่ การอ่านค่า และการลบคุกกี้ 
เรียนรู้การใช้ new Date() เพื่อควบคุมอายุคุกกี้ และการตรวจสอบผลลัพธ์ใน console เพื่อดูการเปลี่ยนแปลงแบบเรียลไทม์

ขั้นต่อมา ฝึกการเชื่อมโยงโค้ดด้วย ES Module โดยใช้ import/export ระหว่างไฟล์ เพื่อเข้าใจการทำงานของ type="module" ใน index.html 
และความแตกต่างของ CommonJS กับ ES Module รวมถึงฝึกดีบักเมื่อเกิดปัญหา เช่น require is not defined ในไฟล์โมดูล

เมื่อเข้าใจโครงสร้างทั้งหมดแล้ว ฝึกเขียนโค้ดให้สะอาดและเป็นระบบมากขึ้น โดยเพิ่มการ log วันที่หมดอายุและสถานะของคุกกี้ทุกครั้งที่มีการเปลี่ยนแปลง 
เพื่อใช้ตรวจสอบผลการทำงานและเพิ่มความเข้าใจเรื่อง life cycle ของคุกกี้ในเบราว์เซอร์

สุดท้าย สรุปสิ่งที่ได้เรียนรู้จากโปรเจกต์นี้คือ การทำงานของ document.cookie การใช้ static methods ในคลาส JavaScript การประยุกต์ Date object 
กับ expires การเข้ารหัสค่าในคุกกี้ การเขียนโมดูลเชื่อมกัน และการทดสอบผลด้วย console.log
