สิ่งที่ได้เรียนรู้จาก Progression 26_sep_25.js , 27_sep_25.js

1. การสร้างลำดับ (Sequence)
- สร้างได้ 2 วิธีหลัก
  1) Array methods เช่น Array.from({ length: n }, (_, i) => … ) → โค้ดสั้น กระชับ
  2) for loop → เห็นขั้นตอนชัด คำนวณ term ทีละตัว และบวก sum ไปพร้อมกัน

2. สูตรคณิตศาสตร์ที่เชื่อมกับโค้ด
Arithmetic Progression (AP):
- Term: a1 + i*d
- Sum: S_n = n/2 * [2a1 + (n-1)d]

Geometric Progression (GP):
- Term: a1 * r^i
- Sum: S_n = a1 * (r^n - 1) / (r - 1)

3. การใช้ Object เก็บผลลัพธ์
- return { terms, sum }
  terms = ลิสต์ของลำดับ
  sum   = ผลรวมทั้งหมด

4. ความยืดหยุ่นของโค้ด
- ฟังก์ชัน progression(type, a1, step, n)
- เลือกได้ว่าจะเป็น "AP" หรือ "GP"
- ตรวจสอบ error ได้ถ้า type ไม่ถูกต้อง

สรุป
- ได้ฝึกทั้ง Array method และ for loop → เข้าใจทั้งแบบกระชับและแบบละเอียด
- ใช้สูตรคณิตศาสตร์ (AP & GP) เชื่อมกับโค้ดจริง
- เรียนรู้การออกแบบฟังก์ชันให้ยืดหยุ่น, การใช้ object เก็บข้อมูลและการคิดเชิงคำนวณ
- for loop → เห็นภาพทีละขั้นชัดเจน
- Array method → โค้ดสั้นเข้าใจง่าย
- ใช้ทั้งสองแบบร่วมกัน → เข้าใจได้ลึกและเลือกใช้ได้ตามสถานการณ์
