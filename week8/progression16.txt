วันนี้ได้เรียนรู้เกี่ยวกับการตรวจสอบข้อมูลในอาร์เรย์ (Data Validation) 
โดยใช้เมธอด filter() เพื่อกรองเฉพาะข้อมูลที่ตรงตามเงื่อนไข เช่น
- ตรวจสอบชนิดข้อมูลด้วย typeof
- ใช้ Number.isFinite() เพื่อเช็กว่าค่าที่เป็นตัวเลขไม่เป็น Infinity หรือ NaN
- ตรวจสอบเงื่อนไขเชิงตรรกะ เช่น age > 1 และ vaccinated === true
นอกจากนี้ยังได้เข้าใจแนวคิดเรื่องการป้องกัน error โดยการเช็กว่า input 
เป็นอาร์เรย์จริง ๆ ก่อนทำงาน (ด้วย Array.isArray)
สรุปคือ ฟังก์ชันนี้ช่วยให้เข้าใจการทำ Data Filtering และ Type Checking 
ใน JavaScript ได้ชัดเจนยิ่งขึ้น
และ ได่เห็นภาพของการนำ array มาใส่ในcondition if else มากขึ้น
