พื้นฐาน Array (เพิ่มจาก loop เดิม)

ฝึก push / pop (ท้าย) และ shift / unshift (ต้น)

เข้าใจว่าเมธอดพวกนี้ เปลี่ยนอาเรย์ต้นฉบับ (mutable)

เลือกสมาชิกออกมาหรือแทนที่

splice → ลบ / แทรก / แทนค่า (แก้ของเดิม)

slice → คัดลอกบางส่วน (ไม่แก้ของเดิม)

fill → แทนค่าทั้งอาเรย์ด้วยค่าเดียว

การแปลงค่าและคัดกรอง

map → แปลงค่าทุกสมาชิกเป็นแบบใหม่

filter → เลือกเฉพาะสมาชิกที่ตรงเงื่อนไข

ทั้งคู่สร้าง อาเรย์ใหม่ ไม่แก้ของเดิม

การรวมค่า

reduce → ย่ออาเรย์ให้เหลือค่าเดียว เช่น ผลรวม, คูณ, ค่าเฉลี่ย

ฝึกใช้ ค่าเริ่มต้น (initial value)

การตรวจสอบและค้นหา

some → มีสมาชิกใดอย่างน้อย 1 ตรงเงื่อนไขไหม

includes → ตรวจสอบว่าอาเรย์มีค่านั้น หรือสตริงมีคำนั้น

การจัดเรียงและการแปลงรูปแบบ

sort → เรียงข้อมูล (เข้าใจว่ามันแก้ต้นฉบับด้วย)

ใช้ map + join เพื่อเปลี่ยนรูปแบบ เช่น แปลงเป็นตัวพิมพ์ใหญ่แล้วรวมเป็นสตริงเดียว

ความก้าวหน้า

จาก อาทิตย์ที่แล้ว → โฟกัสที่ loop + condition + การตรวจสอบเงื่อนไขเบื้องต้น

มา อาทิตย์นี้ → ต่อยอดไปที่ Array และฟังก์ชันของมัน (map, filter, reduce, sort ฯลฯ)

ถือว่าเริ่มจาก “วนลูปเอง” → มาถึง “ใช้เมธอดสำเร็จรูปที่ช่วยลดงานเขียน loop”
