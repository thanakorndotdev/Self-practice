1. Factorial (พื้นฐาน loop และ condition)

เรียนรู้การใช้ if-else เพื่อตรวจสอบเงื่อนไข เช่น n < 0, n = 0, n > 0

ฝึกการวนลูป for เพื่อคำนวณค่า n! (แฟกทอเรียล)

สังเกตการตรวจสอบว่าเป็น integer ด้วย Number.isInteger()

2. นับสระ (เพิ่ม logic + switch-case)

ฝึกตรวจสอบ ชนิดข้อมูล ด้วย typeof

ใช้ loop เดินทุกตัวอักษรในสตริง

ใช้ switch-case เพื่อตรวจสอบว่าตัวอักษรเป็นสระ (A, E, I, O, U) หรือไม่

ฝึกใช้การ แปลงตัวอักษรเป็นพิมพ์ใหญ่ เพื่อไม่ต้องแยกเคสเล็ก/ใหญ่

3. การตรวจสอบจำนวนเฉพาะ (Prime Number)

เข้าใจความหมายของ จำนวนเฉพาะ (มากกว่า 1 และหารลงตัวได้แค่ 1 กับตัวมันเอง)

ใช้เงื่อนไขตรวจสอบ ชนิดข้อมูลและค่า

ใช้ nested loop ตรวจสอบการหารลงตัวตั้งแต่ 2 จนถึง n-1

4. การพิมพ์รูปแบบ (Pattern Printing)

เรียนรู้การใช้ loop สร้างรูปแบบการแสดงผล

ใช้ repeat() ทำซ้ำตัวอักษรตามจำนวนที่กำหนด

ฝึกการจัดรูปแบบเอาท์พุตด้วยการวนลูป

5. พื้นฐานการเข้าถึงและแก้ไขอาเรย์

ใช้ push และ pop สำหรับการเพิ่ม/ลบที่ท้ายอาเรย์

ใช้ unshift และ shift สำหรับการเพิ่ม/ลบที่ต้นอาเรย์

เข้าใจว่าเป็นการ เปลี่ยนอาเรย์ต้นฉบับ (mutable)

6. เลือกสมาชิกออกมาหรือแทนที่

ใช้ splice เพื่อลบ/แทนที่/แทรกสมาชิกในอาเรย์ (แก้ของเดิม)

ใช้ slice เพื่อคัดลอกช่วงของอาเรย์ (ไม่แก้ของเดิม)

ใช้ fill เพื่อแทนที่ค่าทั้งหมดในอาเรย์ด้วยค่าเดียวกัน

7. การแปลงค่าและคัดกรอง

ใช้ map เพื่อแปลงค่าทุกสมาชิกในอาเรย์

ใช้ filter เพื่อคัดกรองเฉพาะสมาชิกที่ผ่านเงื่อนไข

ทั้งสองเมธอดจะสร้าง อาเรย์ใหม่โดยไม่แก้ของเดิม

8. การรวมค่า

ใช้ reduce เพื่อลดอาเรย์ลงเป็นค่าเดียว เช่น ผลรวม, คูณ, ค่าเฉลี่ย

เรียนรู้การใช้ค่าเริ่มต้นใน reduce

9. การตรวจสอบและค้นหา

ใช้ some เพื่อตรวจสอบว่ามีอย่างน้อย 1 สมาชิกตรงเงื่อนไขหรือไม่

ใช้ includes เพื่อตรวจสอบว่าอาเรย์มีค่าที่กำหนด หรือสตริงมีคำย่อยที่กำหนด

10. การจัดเรียงและแปลงรูปแบบ

ใช้ sort เพื่อเรียงสมาชิก (เข้าใจว่า sort จะ แก้อาเรย์ต้นฉบับ)

ใช้ map ร่วมกับ join เพื่อเปลี่ยนรูปแบบการแสดงผล เช่น เปลี่ยนเป็นตัวพิมพ์ใหญ่และเชื่อมเป็นสตริงเดียว
